대충 훓어보는 식으로 읽어보려 했는데도 700페이지짜리라 시간이 좀 걸리네…
밑줄치며 일단 하루만에 1독을 하려 했으나, 이틀이나 지났다 벌써… 스터디 알고리즘 문제 푸는것과 병행하며 해야겠다. 
우선, 남산도서관에서 알고리즘 관련 도서 몇권 읽어서 그런지 용어는 처음 읽을때 처럼 낯설지는 않다. 
일단 용어에 익숙해지고, 용어가 어떤걸 의미하는지 알아두고, 그 용어별로 어떤 상황에 사용되는지… 마지막으로 그걸 어떻게 구현하는지가 최종 목표...
최소 10독 이상을 목표로 하고있는데… 문제는 보지도않고 이론위주로만 훓어보는데도 이틀이나 걸렸다…
일단 읽으며 최대한 익숙해지고 최소한 알고리즘 스터디에 필요한 내용들은 메모해두고 챙겨봐야겠다...
